Preset dropdown (Recommended)

A Validation Type dropdown with presets: None, Email, Postal Code Prefix (A1A), Custom Regex. Selecting a preset fills in rules automatically; only "Custom Regex" reveals manual fields. Simple for non-technical admins, extensible later.

Manual freeform fields

Always show max_length, regex, and normalize toggle. No presets — admin fills in everything manually. More flexible but requires regex knowledge and is error-prone.
