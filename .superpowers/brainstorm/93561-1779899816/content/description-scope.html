Description Field Scope

Where should the description/helper-text field be available?

A

All question types (Recommended)

Add description to every question type's options. Useful for multiple choice instructions, section header clarifications, etc. The postal code privacy disclaimer is just the first use case. Unified data model — one field on every question.

B

Short answer only

Add description only to short_answer type's options. Keeps the change minimal — only touches what's needed for the postal code use case. Other question types can add descriptions later if needed.

What are the first three characters of your postal code?
We ask for the first three characters of your postal code to get a general sense of where responses are coming from. This does not reveal your exact address and is only used for overall geographic analysis.
